# enumerate()

对于一个可迭代的（iterable）/ 可遍历的对象（如列表、字符串），enumerate 将其组成一个索引序列，利用它可以同时获得索引和值

enumerate 多用于在 for 循环中得到计数


enumerate(sequence, [start=0])

参数

  • sequence -- 一个序列、迭代器或其他支持迭代对象。
  • start -- 下标起始位置。

返回值

返回 enumerate (枚举) 对象。
