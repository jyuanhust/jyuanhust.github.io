# 记录目的

之所以要记录博客站点（包括 github 上的和电脑本地的）的修改记录，实在是因为本人比较懒惰，记忆力又不好，往往之前做过的修改过一段时间没过来看又忘记了之前干过什么，甚至文件夹保存在磁盘上哪个位置都忘记了。

# 备份
