# la

la 概述
la 是 risc-v 指令集中的一条伪指令。伪指令，一般会被汇编器翻译成一条或者多条等价的实际指令。伪指令存在的意义就类似于 C 语言中的 typedef 或者 #define 等，完全是出于方便程序员编程、阅读源代码的目的，它本身并没有扩展整个指令集的表达能力。

la (load address), 顾名思义，它是一条地址加载指令。
